# 全新PHP程序开发在线工单管理系统源码

安装环境要求:
服务器空间需要支持 php5.6-7.1 + mysql
1、源码包完整上传至空间，并解压
2、执行 http://域名/ install 根据提示完成安装
3、安装完成，其它的自己调整和设置
项目说明:
